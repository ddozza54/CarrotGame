## 🥕 당근을 찾아라! Game



### 📌 프로젝트 소개

제한된 시간 내에 당근을 모두 클릭하세요!🥕
함께 있는 벌레를 클릭하면 Game Over 되니 조심하세요.🪳

----


### 🖥️ 사용한 기술

- HTML
- CSS
- Vanila Javascript

----


### 🖱️ 주요 기능


- Play & Pause Button 클릭 시 게임이 시작되고 멈춤
- 게임이 시작되면 10초 카운트 다운이 시작됨
- 당근을 클릭 시 당근이 count 됨
- 벌레를 클릭 시 게임이 종료됨
- BGM 을 비롯한 상황 별 사운드 삽입

----
